보유기술

아래와 같은 기술을 활용하여 업무에 적용할 수 있습니다.

Unity

유니티 엔진의 기본 기능 및 스크립트 작업을 통해 빌드를 진행할 수 있습니다.

C#

C# 문법을 활용하여 로직을 구현하고 deligate를 활용한 이벤트 처리를 할 수 있습니다.

Java

Java 문법을 활용하여 웹 개발을 할 수 있을겁니다.

Spring

Spring 프레임워크를 이용해 웹 백엔드 처리를 할 수 있을겁니다.

React

React 프레임워크를 이용해 웹 프론트엔드 처리를 할 수 있을겁니다.

AWS

AWS 클라우드 서버를 활용할 수 있을겁니다.

MySQL

MySql DB를 활용해 백엔드를 구성할 수 있습니다. 당연히 그 과정에서 기초 트랜잭션 문법을 활용하죠.
